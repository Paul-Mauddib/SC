# First Forty-Eight

Marketing site for First Forty-Eight, a retained incident readiness and response practice for protocol and product teams.

Single-file static site: `index.html` (no build step, no dependencies; fonts load from Google Fonts).

## Deploy

**GitHub Pages:** Settings > Pages > Deploy from branch > `main` / root.

**Vercel:** import this repo at vercel.com/new; no framework preset needed.
